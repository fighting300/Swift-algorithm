# Swift-algorithm

### 0.大O表示法
大O | 名称 | 描述
-----| -------| -----------
O(1)          |常数级           |例如:基于索引取出数组中对应的元素，最好的
O(log n)    |对数级           |例如:二分查找，相当好
O(n)          |线性级           |例如:线性查找，还不错
O(nlog n)  |线性对数级    |例如:最快的通用排序算饭，还可以
O(n^2)      |二次方级        |例如:循环套循环的算法，插入排序，有点慢
O(n^3)      |三次方级        |例如:矩阵乘法，特别慢
O(2^n)      |指数级            |例如:旅行商问题，超级慢
O(n!)         |阶乘级            |例如:二分查找
